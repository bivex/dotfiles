### Current `~/.skhdrc`

- `alt + h/j/k/l` — focus west/south/north/east
- `shift + alt + h/j/k/l` — swap west/south/north/east
- `ctrl + alt + h/j/k/l` — resize window
- `alt + f` — toggle zoom fullscreen
- `shift + alt + space` — toggle float
- `alt + 0` — rebalance space
